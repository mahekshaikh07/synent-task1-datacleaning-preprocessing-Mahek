# Task 1: Data Cleaning & Preprocessing

## Problem Statement
Raw data is messy. This project cleans the Titanic dataset to make it ready for analysis.

## Dataset
- Source: [Kaggle - Titanic Dataset](https://www.kaggle.com/datasets/yasserh/titanic-dataset)
- Rows: 891 | Columns: 12

## Approach
1. Loaded and inspected the dataset
2. Handled missing values (Age, Embarked, Cabin)
3. Removed duplicate records
4. Converted data types for Survived and Pclass
5. Renamed columns for consistency

## Results
- No missing values in final dataset
- Clean, consistently named columns
- Saved as `titanic_cleaned.csv`
